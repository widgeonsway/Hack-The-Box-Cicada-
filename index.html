Timeline to Resolution: Cicada

Introduction

Cicada is an easy Windows machine created by Hack The Box.
Tools you will need for this machine are:

  • Hack The Box Account
  • VirtualBox
  • A Kali VM

1

What is Hack The Box?

Hack The Box is a cybersecurity platform that focuses on black box penetration testing. You can visit it here!

2

What is VirtualBox?

VirtualBox is a type 2 hypervisor which allows you to virtualize an operating system such as the one we will be using in this tutorial, Kali Linux.
You can download VirtualBox for your OS here.

3

How do I get connected?

Once you have downloaded VirtualBox and installed a Kali VM:

  1. Log in or sign up for a Hack the Box account.
  2. Click connect to Hack the Box
  3. Click Machines and download your personal openvpn file
  4. Start your target machine
  5. Connect to your openvpn
